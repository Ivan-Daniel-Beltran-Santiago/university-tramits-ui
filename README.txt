Frontend interface for a university procedures management system. Built to facilitate student access to institutional services.

Technologies:

- HTML5
- CSS3
- JavaScript

Features:

- Clean and responsive UI
- Simple navigation for different types of student requests

Getting Started:

Clone the repository and open index.html in your browser.

git clone https://github.com/Ivan-Daniel-Beltran-Santiago/diagramasion.git
